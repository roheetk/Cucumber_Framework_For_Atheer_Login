## Cucumber Framework for Atheer Login Automation

This project automates the Login functionality of Atheer web application using Selenium WebDriver, Java, and BDD Cucumber. 

## Tech Stack

- Java 23.0.1
- Selenium WebDriver
- Cucumber BDD
- Maven
- TestNG
- Extent
- GitHub version control
---
## Framework Features

✅ Page Object Model (POM)  
✅ Centralized Configuration  
✅ Tag-based Test Execution (@Smoke, @Regression)  
✅ HTML Reporting  
✅ Jenkins CI/CD friendly
